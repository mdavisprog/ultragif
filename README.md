# UltraGIF
Latest release is [v0.2.0](https://github.com/mdavisprog/ultragif/releases/tag/v0.2.0).

[Wiki](https://github.com/mdavisprog/ultragif/wiki).

UltraGIF is an application to view, edit, and combine multiple GIFs into a single GIF. Each loaded
GIF can be organized on the canvas. Each frame's delay can be changed to alter the timings for each
animated GIF in the scene.

![Example](https://github.com/mdavisprog/ultragif/wiki/images/example.gif)

The above scene is exported as:

![Example_Export](https://github.com/mdavisprog/ultragif/wiki/images/example_export.gif)

This application is in the early stages of development, so there are a lot of features not implemented
yet but will be in the near future. The wiki contains more information on how to use this application.
